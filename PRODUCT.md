# Product

## Register

brand

## Users
Potential employers, academic mentors, and fellow developers looking for an ambitious IT Student majoring in Information Security who is learning full-stack development, leverages modern AI tools, and builds systems.

## Product Purpose
To showcase Aristaeus (Darktodoom123) as a detail-oriented, learning developer and IT student. The profile highlights active contributions, key group projects (like Concord HMS and BPM-SYSTEM), and technology exploration with an engaging, interactive presentation.

## Brand Personality
* **Three-word personality**: Curious, Celestial, Tech-Explorer
* **Tone**: Enthusiastic, learning-focused, cosmic (astrological/crab theme), and clean.
* **Emotional goals**: Trust, curiosity, potential.

## Anti-references
* Standard SaaS light-cream card grids (e.g. standard Tailwind templates).
* Generic "minimalist" white portfolios with no color or character.
* Excessive tiny uppercase eyebrows on every section.
* Plain, un-animated project lists.

## Design Principles
* **Celestial Depth**: Use rich, dark oklch purples, indigos, and deep space gradients with glowing interactive accents to create an immersive, cosmic environment.
* **Showcase the Craft**: Present code contributions, stats, and project details with interactive, stateful components that feel alive when hovered or clicked.
* **Performance First**: The site must load instantly, use semantic HTML, and perform animations smoothly using CSS hardware acceleration.
* **Subtle Cosmic Accents**: Incorporate a custom Crab/Cancer constellation or astrology-themed design element as a signature visual mark, matching the user's avatar.

## Accessibility & Inclusion
* High contrast (minimum 4.5:1 ratio) for all text against dark, glowing gradients.
* Respect `prefers-reduced-motion` settings for all transition effects.
* Fully keyboard-navigable interactive components (like project tabs/filters).
